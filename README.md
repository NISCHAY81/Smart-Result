# Smart-Result  
> A student result management / display system built using Spring Boot (Java) & Thymeleaf.

Smart-Result allows academic institutions to manage and display student results in a modern web interface. It employs Spring Boot for back-end services and Thymeleaf for server-side rendering of HTML views.  
Key features include:  
- Entry of subject codes, marks, grades, etc.  
- Calculation/aggregation of results (e.g., SGPA, CGPA) for display.  
- Responsive interface for students and administrators.
